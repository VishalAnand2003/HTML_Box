Box Model

The CSS box model is essentially a box that wraps around every HTML element. It consists of:

Border
In the box the red color outline form the four sides is called border.
Padding
The space between the border and the content is called padding.
Margins
As we can see there is some gap between the upper written h1 tag material and the box that is called margin , it is from all the sides.
Actual content
The wrritten material inside the box is content.

About CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML.CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

The name cascading comes from the specified priority scheme to determine which style rule applies if more than one rule matches a particular element. This cascading priority scheme is predictable.
